Demo


These sample applications are not anything you haven't seen before. The interesting thing is this Single Page App (SPA) is a set of generic views configured with different UI-models.

  • To Do
  • AddressBook
  • Wine cellar

For each Backbone model, a single UI-model defines all views. See the UI model for the sample apps: To Do, Addressbook, Wine Cellar. [Hide]

The data for these demos is stored in your browser local storage (using Backbone.localStorage).

Evolutility.js is still a work in progress.
